Detailed Description

A number of inevitable base classes for making common things like a struct sockaddr easily managable.

Here, we gather a number of helper classes for hiding UN*X API complexity behind an easy to use interface, e.g. for manging memory access via class cmemory. This also contains the main I/O service defined in ROFL that distributes events throughout an OpenFlow protocol endpoint.
