void cthread::start ( )

Starts worker thread.

Sets run flag run_rx_thread to true and calls syscall pthread_create() for starting worker thread.

void cthread::stop ( )

Stops worker thread.

Stops worker thread by setting variable run_rx_thread to false and waking up the thread. Waits for thread termination via pthread_join(). If joining the thread fails, syscall pthread_cancel() destroys ultimately the thread.
